===EGG COLOR ALGORITHM===

0. Create egg_count variable to limit the amount of each egg, so that we 
don't have more eggs than pairs of matches.
1. Reads the assets/egg directory for the different kinds of eggs.
2. Use a directory for storing both the egg sprite, and the amount of 
eggs hidden.

3. WHILE egg_count is below max...
	3.A) Create Card object
	3.B) Pick a random egg_color from options-list
	3.C) Assign egg_color to new Card
	3.D) Increase egg_count and remove egg_color from options-list
	


===CHICKEN POSITION ALGORITHM===
1. Check for filled positions by measuring the length of the coop_row
array, then by the next_box. (y, then x within y...
... I think this is called a linked list?)
	1.A) If coop_row has a length of 2 (starting from 0) 
	skip row.
	1.B) If coop_row's length is less than 2, then insert Chicken
	(increasing the length).
	
2. Set chicken's internal X and Y according to their position in 
the grid: [
		[chicken1, chicken2, ...] # Row one
		[] # Row two
		[] # ...
	]
	
3. Check if coop_list's length is 8 (including 0)
	3.A) True: algorithm is finished, return coop_list
	3.B) Repeat from Step 1.
